# Excel拼音

用于在Excel中将汉字换为拼音的宏

## 使用

将这个宏加入Excel

使用函数getpy将字符串转化为拼音

使用函数getszm将字符串转化为拼音首字母

拼音取常用拼音，多音字可能会出现错误

拼音之间使用空格分隔

### e.g.

=getpy("宽宽的作品")

->kuan kuan de zuo pin 

=getszm("宽宽的作品")

->kkdzp
